# Salamia: Grassroots Rising

A political simulation set in the fictional Federal Republic of Salamia.
You begin as a frustrated neighbourhood volunteer and rise — or fall —
through ward, state, and national politics, fighting to pass the Local
Revenue Empowerment Act (10% of national sales tax devolved to local
bodies).

This is a Godot 4 / GDScript project.

## Current state

The first playable spike lives on branch
[`claude/godot-political-game-dDIJI`](../../tree/claude/godot-political-game-dDIJI).
It covers the **Aryavarta** state, **Volunteer → Ward Councillor** arc.
See that branch's README for systems, content, and run instructions.
